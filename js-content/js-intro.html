What is JavaScript?
JavaScript is a scripting language ( type of programming language that is interpreted by the browser or another runtime environment at runtime) that is used to create website and Web Applications. Most of the website is now use to create interactive website. JavaScript can be run on both clint side (brower side) or server side(with the help of framework like node.js).
What JavaScript can do?
We can use JavaScript to store information, upload or update data or style dynamically (interact with the DOM), handle events and many more things.
Version of JavaScript
There are many versions of JavaScript we call these versions ES(ECMAScript)numbering. Most famous version of JavaScript is ES6.
  1. ES1: The first version of JavaScript is released in 1997. It laid the foundation for language's syntax and core features.
  2. ES2: The second version of JavaScript is released in 1998.
  3. ES3: The third version of JavaScript is released in 1999.
  4. ES4: This version was abandoned due to disagreements within the ECMAScript committee over its proposed features.
  5. ES5: The fifth version of JavaScript is released in 2009. ES5 introduced several new features such as strict mode, JSON support, and additional array methods like map, filter, and reduce.
  6. ES6: The sixth version of JavaScript is released in 2015. ES6 brought major enhancements to the language, including let and const for variable declarations, arrow functions, classes, template literals, destructuring assignment, and promises.
